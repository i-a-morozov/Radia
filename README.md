# Radia
3D Magnetostatics Computer Code

For copyright/licensing look at the original Radia repository

# Linux compilation

The following assumes standard WM installation path.
Replace WM version in the `Makefile_linux`.

```bash
$ git clone https://github.com/i-a-morozov/Radia
$ cd Radia/cpp/gcc && make -f Makefile_linux
$ cp Radia.exe  ../../env/radia_math/Radia

```

# Installation

```bash
$ cp -r ./env/radia_math/Radia ...

```
